### Installation

```console
# clone the repo
$ git clone https://github.com/xadhrit/xira.git

# change the working directory to xira
$ cd xira

# install the requirements
$ python3 -m pip install -r requirements.txt

```

### Usage

```console
$ python3 xira.py

Enter the target's url : https://www.example.com
```
